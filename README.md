# 🍽️ Crave - Recipe Finder

A simple web application that displays recipes based on different cuisines using the **TheMealDB API**.
Users can explore meals from various countries and view them in a clean card-based layout.

---

## 📌 Overview

Cuisine Recipe Finder fetches meal data from an external API and displays it as recipe cards.

Each card includes:

* Meal image
* Meal name
* Cuisine type

The app dynamically loads recipes from multiple cuisines and renders them on the page using **JavaScript DOM manipulation**.

---

## 🚀 Features

* 🖼️ Card-based UI for each recipe
* ⚡ Data fetched dynamically from an API
* 📱 Responsive layout
* 🔄 Asynchronous data fetching with `fetch()` and `async/await`

---

## 🛠️ Built With

* **HTML5**
* **CSS3**
* **JavaScript (ES6)**
* **TheMealDB API**
